# lame server

A single self-contained page: ~1,500 glyph particles morph between **lame**,
**server**, and **lame server** (two lines), with a wave, a spiral, and a
Lissajous curve in between. The cursor repels particles; `prefers-reduced-motion`
holds the two-line form still.

No dependencies, no network requests — the font ships inside the HTML as a
base64 data URI.

## Run

Open `index.html` in a browser. That's it.

## Type

TeX Gyre Pagella Bold (GUST Font License), subset to the glyphs the page uses
and embedded as woff2.
